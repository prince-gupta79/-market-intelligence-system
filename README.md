Market Intelligence System :

I built this during Class 12 while teaching myself Python and data science. Started as a simple price alert script and kept adding things until it became... this.
It monitors stock prices in real time, runs ML models to detect trends, scrapes financial news and analyses sentiment, then sends everything to my phone via Telegram. Fully automated.


What it actually does

- Tracks live prices for NVDA, NEE, GLD, BTC-USD using yfinance
- Logs everything to a local SQLite database with timestamps
- Calculates Simple Moving Average crossovers to generate buy/sell signals
- Backtests the strategy against 1 year of historical data to see if it actually works
- Runs 98 different SMA window simulations to find the optimal one
- Uses Random Forest and Linear Regression to predict prices
- Scrapes Google News RSS feeds and runs TextBlob sentiment analysis on headlines
- Combines price trend + news sentiment into a single morning report
- Sends alerts and daily briefings directly to Telegram

Setup
bashgit clone https://github.com/prince-gupta79/market-intelligence-system.git
cd market-intelligence-system
pip install -r requirements.txt
For Telegram alerts, create a .env file:
BOT_TOKEN=your_token
CHAT_ID=your_chat_id
Then run:
bashpython 11_master.sentinel.py

What I learned
Honestly the biggest thing was learning how to connect all the pieces together — getting live data, storing it, running models on it, and then actually doing something useful with the output instead of just printing results to a terminal.
Also learned that backtesting a strategy and getting a good result doesn't mean it works in real markets. The optimization engine taught me that overfitting is a real problem.

Note
This is a learning project. Not financial advice, not a real trading system. Don't use it to make actual trades.
